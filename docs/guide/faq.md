# 常见问题

## 同花顺连接失败

**问题**: 无法连接到同花顺客户端

**解决方案**:
1. 确保同花顺客户端已启动
2. 检查版本是否为 10.4 或更高
3. 以管理员权限运行 EasyTHS

## Redis 连接错误

**问题**: 无法连接到 Redis 服务

**解决方案**:
1. 确保 Redis 服务已启动
2. 检查 `config.yaml` 中的 Redis 配置
3. 使用 `redis-cli ping` 测试连接
