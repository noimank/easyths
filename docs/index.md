# EasyTHS

**同花顺交易自动化系统** - 基于 Python 的同花顺客户端自动化交易工具。

## 特性

- 🚀 **自动化交易** - 自动执行交易策略，无需人工干预
- 🖥️ **GUI 自动化** - 基于 pywinauto 的 Windows GUI 自动化
- 🌐 **RESTful API** - FastAPI 提供的 HTTP API 接口
- 📊 **数据管理** - 支持交易数据的记录和查询
- 🔒 **安全可靠** - 本地运行，数据不上传第三方服务器

## 快速开始

```bash
# 安装
pip install easyths

# 启动服务
easyths
```
