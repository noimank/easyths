# 贡献指南

感谢你对 EasyTHS 的关注！

## 如何贡献

1. Fork 本仓库
2. 创建特性分支 (`git checkout -b feature/AmazingFeature`)
3. 提交更改 (`git commit -m 'Add some AmazingFeature'`)
4. 推送到分支 (`git push origin feature/AmazingFeature`)
5. 提交 Pull Request

## 代码规范

- 使用 Black 进行代码格式化
- 使用 Ruff 进行代码检查
- 编写单元测试
- 更新相关文档

## 报告问题

请在 [Issues](https://github.com/noimank/easyths/issues) 中报告问题。
