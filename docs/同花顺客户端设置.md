# 同花顺交易客户端设置指南

为了确保 QuantTrader 自动化交易系统能够正常运行，您需要对同花顺交易客户端进行一些必要的设置。请按照以下步骤进行配置。

## 🚨 重要提醒

- **必须设置**：以下所有设置项都是必须的，缺少任何一项都可能导致程序无法正常运行
- **设置位置**：同花顺交易客户端 → 系统设置
- **重启生效**：部分设置需要重启同花顺客户端才能生效

## 1. 关闭悬浮工具栏

### 设置位置
设置 → 界面设置 → 显示悬浮工具栏

### 操作步骤
1. 打开同花顺交易客户端
2. 点击顶部菜单"工具"
3. 选择"设置"
4. 在设置对话框中，切换到"界面设置"或"常规设置"选项卡
5. 找到"显示悬浮工具栏"选项
6. **取消勾选**该选项
7. 点击"确定"保存设置

### ⚠️ 为什么必须关闭？
悬浮工具栏会干扰程序对弹窗的判断，导致程序无法正确识别操作结果，从而影响自动化流程的正常执行。

## 2. 优化交易确认设置

### 设置位置
系统设置 → 快速交易

### 2.1 关闭买入确认
- **选项**：买入时是否需要确认
- **设置**：**选择"否"**
- **作用**：避免每次买入时弹出确认对话框，提高交易速度

### 2.2 关闭卖出确认
- **选项**：卖出时是否需要确认
- **设置**：**选择"否"**
- **作用**：避免每次卖出时弹出确认对话框，提高交易速度

### 2.3 关闭撤单确认
- **选项**：撤单时是否需要确认
- **设置**：**选择"否"**
- **作用**：加快撤单操作速度

### 2.4 关闭底部提示
- **选项**：底部是否提示委托或成交信息
- **设置**：**选择"否"**
- **作用**：避免提示信息干扰程序判断

### 2.5 开启页面切换清空
- **选项**：切换页面是否清空代码
- **设置**：**选择"是"**
- **作用**：切换页面时自动清空股票代码，避免代码残留影响下一次操作

## 3. 清空默认价格设置

### 设置位置
系统设置 → 交易设置

### 3.1 清空默认买入价格
- **选项**：默认买入价格
- **设置**：**留空或设为"0"**
- **作用**：避免程序误用默认价格

### 3.2 清空默认卖出价格
- **选项**：默认卖出价格
- **设置**：**留空或设为"0"**
- **作用**：确保每次交易都输入准确的卖出价格

## 4. 其他推荐设置

### 4.1 委托数量设置（可选）
- **选项**：默认委托数量
- **建议**：设为 "100" 或留空
- **作用**：A股最小交易单位为100股

### 4.2 快捷键设置（可选）
- 建议熟悉同花顺的快捷键
- F1：买入
- F2：卖出
- F4：查询
- ESC：取消/退出

## 5. 验证设置是否生效

### 5.1 测试买入操作
1. 按下 F1 键打开买入界面
2. 输入任意6位股票代码
3. 输入价格和数量
4. 点击"买入"按钮
5. **应该直接成交**，不弹出确认对话框

### 5.2 测试页面切换
1. 在买入界面输入股票代码
2. 按下 F2 切换到卖出界面
3. **股票代码应该自动清空**

### 5.3 检查悬浮工具栏
- 确保屏幕上没有悬浮的工具栏窗口
- 如果仍然存在，尝试重启同花顺客户端

## 6. 常见问题

### Q1: 找不到某个设置选项？
**A**: 不同版本的同花顺客户端设置位置可能略有不同：
- 尝试在"工具"→"设置"中查找
- 有些版本可能在"系统"→"系统设置"中
- 如果实在找不到，可以联系同花顺客服

### Q2: 设置后没有生效？
**A**: 请尝试：
1. 完全退出同花顺客户端
2. 重新启动程序
3. 再次确认设置项是否已保存

### Q3: 交易安全考虑？
**A**: 虽然关闭了确认对话框可以提高速度，但也增加了误操作的风险：
- 确保程序测试无误后再进行实盘操作
- 建议先在模拟环境测试
- 保持API密钥安全

### Q4: 可以保留部分确认吗？
**A**: 不建议。所有确认对话框都应该关闭，否则会导致程序在等待确认时卡死。

## 7. 设置完成后的流程

1. **验证所有设置**：按照第5节的方法测试
2. **启动 QuantTrader**：运行自动化交易系统
3. **测试连接**：使用系统的健康检查接口确认连接正常
4. **小单测试**：先用小笔交易测试自动化流程
